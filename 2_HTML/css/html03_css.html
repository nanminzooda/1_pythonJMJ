class 선택자와 id 선택자

class 선택자

문서 내에서 여러번 사용될 수 있는 선택자로 클래스를 구분할 때는 점(.)을 이용합니다.

클래스를 선택자로 사용할 경우 요소명을 생략할 수 있습니다.

class 선택자

id 선택자는 중복을 허용하지 않습니다. 요소명과 아이디명을 구분 할때는 파운드 (#) 기호를 사용합니다.

  • 회원가입
  • 사이트맵
  • 쉬어가기
